# Engagement COE 2026 First Half Recap

This version matches the generated artwork as closely as possible by using it as the complete page design and positioning a real video player inside the illustrated screen.

## Video setup

For the most reliable playback:

1. Export the recap as MP4.
2. Rename it to `recap.mp4`.
3. Place it beside `index.html`.

The page also includes the Google Drive file as a fallback source.

## GitHub Pages

Upload:

- `index.html`
- `style.css`
- `script.js`
- `assets/recap-board.png`
- `recap.mp4` if using a local video

Then enable GitHub Pages from the repository settings.

## Mobile

- Portrait preserves the full poster design.
- Landscape switches to an edge-to-edge video player.
